# RunWise Admin

A standalone admin dashboard for RunWise: users, runners, jobs, disputes,
verification, vehicles, wallets, restricted items, and an audit log. Talks
directly to Supabase (same project as the mobile app and the customer web
app) using the anon key plus RLS role checks — no separate API server.

This replaces the admin sections that used to live inline inside the
customer-facing `app.js` SPA. Keeping admin as its own app means:
- Admin code and dependencies never ship to customer/runner browsers
- The admin surface can be deployed on a separate URL/domain if you want
  an extra access boundary
- A dedicated, consistent UI kit (shadcn/ui) instead of hand-rolled tables

## Setup

```bash
cd admin
npm install
cp .env.example .env   # fill in your Supabase project values
npm run dev
```

Required environment variables (see `.env.example`):
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## Auth

Admins sign in with normal Supabase email/password. After sign-in, the app
checks `profiles.role === 'admin'` and signs the user back out if it
doesn't match — so this only works for accounts already marked as admin
in the database.

## Build

```bash
npm run build   # outputs to admin/dist
npm run preview # serve the production build locally
```

## Note on origin

This app was originally generated by a different tool (Replit) as part of
`artifacts/runwise-admin/` on the `replit-build` branch, alongside a
duplicate copy of the mobile app and two scaffold-only folders
(`artifacts/api-server`, `artifacts/mockup-sandbox`). Only this admin app
was worth keeping — the mobile app duplicate was functionally identical to
what's already in `mobile/`, and the other two were confirmed (by the
tool's own notes) to be unused scaffolding. Its `package.json` and
`vite.config.ts` were rewritten here to remove hard dependencies on
Replit's private monorepo tooling (pnpm `catalog:` versions, a
`workspace:*`-only package, Replit-only Vite plugins, and required
`PORT`/`BASE_PATH` env vars that made it crash outside Replit) so it runs
as a normal standalone Vite app anywhere.
